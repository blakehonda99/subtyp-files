How to use Subtyp Storage Windows App

1. Open the prerequisites folder and install winfsp (do not alter the settings for it, use all default settings).

2. Open the rclone folder.

3. Open set-password.bat, then type in your password that has been supplied to you.

4. Open rclone.conf, then type in your username into the 5th line, next to user. It should look like user = USERNAME.

5. After all credentials have been inputted and everything has been installed, all you have to do is double click the run.cmd. This will mount the cloud storage to X:\subtyp-storage. You can access, upload, view, and download all files from here.
